# Banestes Gerenciador de Brindes

Bem-vindo ao **Banestes Gerenciador de Brindes**, uma solução digital criada sob medida para o setor **Núcleo Administrativo** da **Banestes Asset Management**. Este sistema visa modernizar, automatizar e tornar mais transparente todo o processo de controle, requisição, distribuição e rastreamento de brindes corporativos, otimizando recursos e facilitando o dia a dia administrativo.

## ✨ Visão Geral

O **Gerenciador de Brindes** foi desenvolvido com base nas demandas específicas do setor, empregando o **Google Apps Script** para garantir integração nativa com a suíte Google Workspace, facilidade de acesso multiusuários, segurança dos dados e baixos custos operacionais.

## 🎯 Objetivo

Oferecer uma plataforma simples, eficiente e centralizada, permitindo ao *Núcleo Administrativo* do Banestes Asset Management gerenciar brindes de forma ágil e controlada, reduzindo riscos, desperdícios e proporcionando total visibilidade das movimentações.

---

## 🛠️ Tecnologias Utilizadas

- **Google Apps Script**  
  Plataforma usada para criar automações e aplicativos sobre planilhas, documentos e outros serviços Google.
- **Google Sheets**  
  Utilizado como base de dados para cadastro, movimentação e estoque dos brindes.
- **HTML & JavaScript**  
  Estrutura e lógica da interface de usuário, integrando com o Google Apps Script para um sistema web responsivo e leve.

---

## ⚙️ Funcionalidades Detalhadas

- **Cadastro de Brindes**  
  Registre novos brindes, definindo nome, categoria, foto, valor unitário e estoques iniciais.
- **Requisição e Aprovação**  
  Usuários autorizados podem solicitar brindes para eventos, colaboradores ou demandas especiais. Toda solicitação é encaminhada para validação pelo gestor responsável.
- **Controle de Estoque Dinâmico**  
  O saldo dos brindes é atualizado automaticamente a cada movimentação, minimizando erros manuais.
- **Histórico e Auditoria**  
  Todo movimento (entrada, saída, devolução) fica registrado para consultas e auditoria.
- **Relatórios Avançados**  
  Geração de relatórios por período, tipo de brinde, usuário solicitante/aprovador, status de requisições entre outros filtros.
- **Alertas Inteligentes**  
  Notificações automáticas para estoques baixos, aprovações pendentes e movimentações críticas.
- **Permissões & Segurança**  
  Níveis de acesso customizáveis, garantindo que apenas usuários habilitados alterem cadastros ou aprovem solicitações.
- **Interface Intuitiva**  
  Webapp responsivo, acessível via desktop e dispositivos móveis, com menus claros e operações guiadas por passo a passo.

## 🖥️ Como Utilizar

1. **Acesse o WebApp:**  
   Link disponibilizado internamente via Drive, menu ou email corporativo.
2. **Faça Login:**  
   Utilize sua conta Banestes (login Google Workspace).
3. **Menu Principal:**  
   - Cadastro de Brindes  
   - Requisitar Brinde  
   - Consultar Estoque  
   - Aprovar Solicitações  
   - Relatórios  
   - Auditoria
4. **Gerencie as Movimentações:**  
   Siga as instruções apresentadas na tela para cada função. Os campos obrigatórios são marcados com *asteriscos*.

> **Obs.:** Permissões e acesso são gerenciados pelo Administrador do Sistema. Em caso de dúvidas ou necessidade de acesso, contate o Núcleo Administrativo.
